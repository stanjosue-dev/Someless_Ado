HTML Download Button Challenge

Create a single HTML page with CSS written inside the same file.

Your page must include a button that lets the user download an image or a file when they click it.

Student Instructions

  • Use only one HTML file.
  • Write your CSS inside the HTML file using the `style` tag.
  • Add a clear download button.
  • Make the page neat, colorful, and easy to read.
  • Host the project on GitHub Pages.
  • Share the live GitHub Pages link with me.
